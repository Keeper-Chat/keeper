# Security Policy

## Supported Versions

We support both a client and a protocol, and each has its own versioning and support lifecycle. Below you’ll find the versions for both.

### Client

| Version | Supported          |
| ------- | ------------------ |
| 0.1.2   | :white_check_mark: |

### Protocol

| Version | Supported          |
| ------- | ------------------ |
| 0.1.0   | :white_check_mark: |

## Reporting a Vulnerability

If you find a vulnerability then please open a issue on [this repository](https://github.com/Keeper-Chat/keeper).
